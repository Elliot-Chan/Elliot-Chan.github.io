关于函数有界(无界)的讨论

求解思路

  1. 函数的有界性:
    设函数f(x)在数集X上有定义,如果存在常数M,当x ∈ X时,f(x)≤M, 则称f(x)在X上有上界;如果存在常数M,当x ∈ X时,f(x)≥M, 则称f(x)在X上有下界

  2. 有界和无界的若干充分条件

    1. 设$\lim\limits_{x\rightarrow x_0^-}f(x)$存在,则∃δ > 0,当时,f(x)有界;对,结论类似

    2. 设$\lim\limits_{x\rightarrow\infty}f(x)$存在,则∃X > 0,当|x|>X时,f(x)有界;对x → +∞,结论类似

    3. 设f(x)在[a, b]上连续,则f(x)在[a, b]上有界

    4. f(x)有最大(小)值,则f(x)有上(下)界

    5. 有界函数与有界函数之和,积均为有界函数

    6. 设$\lim\limits_{x\rightarrow\square}f(x) = \infty$,则f(x)在 ▫ 的去心邻域内无界

无穷小的比较

知识点

  1. $\lim\limits_{x\rightarrow \square}\frac{\alpha(x)}{\beta(x)} = 0 \Longrightarrow $ α(x) 是 β(x)的高阶无穷小

  2. $\lim\limits_{x\rightarrow \square}\frac{\alpha(x)}{\beta(x)} = C \neq 0 \Longrightarrow $ α(x) 是 β(x)的同阶无穷小

  3. $\lim\limits_{x\rightarrow \square}\frac{\alpha(x)}{\beta(x)} = 1 \Longrightarrow $ α(x) 是 β(x)的等价无穷小

  4. $\lim\limits_{x\rightarrow \square}\frac{\alpha(x)}{\beta^k(x)} = C \neq 0 \Longrightarrow $ α(x) 是 β(x)的k阶无穷小

思路

  1. 洛必达法则

  2. 佩亚诺余项泰勒公式转开

  3. 等价无穷小 条件是▫ → 0

极限运算定理的正确运用

知识点

四则运算法则

设$\lim\limits_{x\rightarrow \square}u(x) \exists = A$, $\lim\limits_{x\rightarrow \square}v(x) \exists = B$

  1. $\lim\limits_{x\rightarrow \square}(u(x)\pm v(x)) = \lim\limits_{x\rightarrow \square}u(x) + \lim\limits_{x\rightarrow \square}v(x) = A \pm B$

  2. $\lim\limits_{x\rightarrow \square}(u(x)v(x)) = \lim\limits_{x\rightarrow \square}u(x)\lim\limits_{x\rightarrow \square}v(x) = AB$

  3. $\lim\limits_{x\rightarrow \square}(cu(x)\pm v(x)) = c\lim\limits_{x\rightarrow \square}u(x) = cA$

  4. $\lim\limits_{x\rightarrow \square}\frac{u(x)}{v(x)} = \frac{\lim\limits_{x\rightarrow \square}u(x)}{\lim\limits_{x\rightarrow \square}v(x)} = \frac{A}{B}$ (设B ≠ 0)

  5. $\lim\limits_{x\rightarrow \square}u(x) = 0$,并设在▫的去心邻域内k(x)有界,则 $\lim\limits_{x\rightarrow \square}k(x)u(x) = 0$
